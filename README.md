react-ghpages-blog
===

想法来源于 [vue-ghpages-blog](https://github.com/viko16/vue-ghpages-blog)

一个基于 GitHub Pages , 使用 GitHub API 的 Blog.

React + Redux + webpack

## Usage
```
git clone https://github.com/zowiegong/react-ghpages-blog.git

cd react-ghpages-blog

npm install
# or
yarn install

修改 src/config.js

npm run build
# or
yarn build
```
